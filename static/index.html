CODACHROME

Codachrome is a little tool that helps you find the next notes to play. Also, it only works on Chrome!

Why do I have to click a magic button?

Chrome has an autoplay policy that impacts when sound can happen, requiring user interaction. Here are their official updates on that. This sounds totally reasonable when you think about how annoying autoplay sound can be, but turns out, MIDI input doesn't count as user interaction! What! Ok! So that's why you need to click the button. This post about web audio gotchas was also helpful for us.
